# Unser Projekt

Linguins ist eine innovative Lernplattform für Linux (Terminal integration), die darauf abzielt, das Erlernen des Linux-Terminals durch interaktive und benutzerfreundliche Methoden zu erleichtern. 
Unsere Plattform bietet eine Vielzahl von Funktionen, darunter Lernpläne und verschiedene Module, interaktive Übungen und Feedback auf die selbständige Perfomance, um den Lernprozess zu optimieren.


## Funktionen
- **Lernpläne und Module**: Benutzer können aus verschiedenen Lernplänen und Modulen wählen, die auf ihren Kenntnisstand und ihre Lernziele zugeschnitten sind.
- **Interaktive Übungen**: Praktische Übungen ermöglichen es den Benutzern, das Gelernte direkt im Terminal anzuwenden.
- **Feedback und Fortschrittsverfolgung**: Benutzer erhalten sofortiges Feedback zu ihren Übungen und können ihren Fortschritt verfolgen.
- **Benutzerfreundliche Oberfläche**: Eine intuitive und ansprechende Benutzeroberfläche sorgt für ein angenehmes Lernerlebnis.
- **Sicherheitsfunktionen**: Datenschutz und Sicherheit der Benutzerdaten haben höchste Priorität.


## Technologie-Stack
- **Frontend**: Vue.js für die Benutzeroberfläche
- **Backend**: Java mit SpringBoot
- **Datenbank**: PostgreSQL für die Speicherung von Benutzerdaten und Lernfortschritten
- **Versionierung**: Git und GitHub für die Quellcodeverwaltung


## Systemanforderungen
- Betriebssystem: Funktioniert auf allen bekannten Betriebssystemen (Windows, macOS, Linux)
- Empfohlener Arbeitsspeicher: Mindestens 4 GB RAM für eine reibungslose Nutzung der Anwendung
- Empfohlene Bildschirmauflösung: Mindestens 1024x768 Pixel für eine optimale Anzeige der Benutzeroberfläche
